# Order Management API Server

## Overview
Spring Boot 기반 주문 관리 API 서버입니다.
상품 주문, 주문 상태 관리, 주문 조회 기능을 제공하는 백엔드 시스템입니다.

## Tech Stack
- Language: Java 17
- Framework: Spring Boot 3
- ORM: Spring Data JPA
- Database: PostgreSQL
- Build Tool: Gradle
- Authentication: JWT
- SCM: Git / Github
